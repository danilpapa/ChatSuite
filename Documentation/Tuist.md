# Tuist

Tuist — инструмента для декларативного описания Xcode-проектов на Swift. Структура проекта модульная и разделена на несколько подпроектов: 

- **API**: Shared ресурсы и протоколы для контракта.
- **Core**: Ядро приложения, включающее сборку зависимостей (HeedAssembly), сервисы (Services) и сетевой слой (Network).
- **ChatClient**: Основное приложение.
- **DesignKit**: Библиотека для UI-компонентов и ресурсов дизайна.

Workspace определяется в файле `Workspace.swift` и генерирует полный Xcode-проект командой `make generate`.
